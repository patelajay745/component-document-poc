## Props

| Name        | Type   | Default | Description                            |
| ----------- | ------ | ------- | -------------------------------------- |
| **title**   | string | -       | The title displayed on the card.       |
| **content** | string | -       | The content displayed below the title. |

The `MyCard` component is a versatile and reusable UI element that can be styled and customized to fit your application's needs.
